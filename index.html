Imperium

Una aplicación de escritorio de Gestion De Stock.

Fue un programa hecho a pedido, comenzo siendo una aplicacion de gestion de stock, pero fue evolucionando a pedido del cliente, abarcando funciones como clientes de cuenta corriente, manejo y control de facturas, tanto fiscal como no. La misma aplicacion fue implementada en 3 locales. Se realizo en el lenguaje de programación Java y con una base de datos basada en MySQL.

  • Fecha: Junio 2015
  • Cliente: Threads
  • Categoria: Programación a pedido
